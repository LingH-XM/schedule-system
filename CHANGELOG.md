# 版本说明

本项目采用语义化版本号。当前版本为 **v0.3.0**。

## v0.3.0 — 2026-07-21

本版本重点完成学校多账户权限体系、登录与平台视觉统一、排课工作台交互优化，以及默认规则和智能求解策略完善。

### 学校账户与权限

- 引入 `schoolId` 作为学校级数据隔离标识，学校数据、排课方案与规则按学校归属访问。
- 学校管理员可新增子账户，并按校区、年级及功能权限分配管理范围。
- 子管理员账号统一使用 `tch` 前缀，账户列表拆分展示角色、校区、年级和权限范围。
- 后端新增身份校验、权限守卫和数据范围校验，限制越权读取、修改和智能排课提交。
- 优化账户停用、回收站、密码重置和登录状态管理。

### 登录与平台视觉

- 重新设计登录页面与校园背景，降低传统门户感，提升现代感和亲和力。
- 建立柔和灰蓝全局语义色板，统一登录页与后台导航、按钮、链接、输入框、开关和选中状态。
- 保留课程色块与成功、警告、错误等业务语义色，避免全局主色覆盖业务状态。
- 优化后端健康检查提示，减少服务启动阶段的误报干扰。

### 排课工作台

- 新增撤回与恢复操作，便于修正手工排课调整。
- 待安排课程支持横向、纵向两种排列模式，默认使用纵向模式并允许用户自由切换。
- 优化待安排课程自适应宽度、剩余课时展示和长课程名称省略处理。
- 缩小节次列和课表无效留白，统一普通课程、活动及不排课卡片的尺寸与居中方式。
- 调整课表单元格高度和课程卡片宽度，改善 14 寸笔记本等小屏幕设备体验。
- 将智能排课与排课日志整合为统一操作流程，求解完成后可直接查看日志。

### 默认规则与智能求解

- 新增“不同天节次分散”默认规则，避免同一教师多天固定在同一节次上课。
- 求解器以教师、节次和星期建立软约束，对跨天重复同节次安排进行评分扣减。
- 新规则兼容主科、副科分类，并同时考虑既有课表和新生成课表。
- 将“默认规则”和“权重分配”归入“高级设置”，增加管理员维护提示，降低误配置风险。

### 其他优化

- 优化使用说明页、后台导航、响应式布局和局部交互状态。
- 新增设计展示页面，用于集中验证平台视觉与交互方案。

### 验证

- 前端生产构建通过。
- NestJS 后端 TypeScript 构建通过。
- OR-Tools 求解器 16 项单元测试全部通过。
- 高级设置、默认规则和权重分配页面完成浏览器交互回归，未发现控制台错误。

## v0.2.0 — 2026-07-20

本版本完成了从基础数据准备、规则配置、智能求解到课表发布与统计导出的完整排课流程，并重点增强了多账户智能排课的稳定性。

### 智能排课与求解器

- 新增智能排课全局 FIFO 队列：多个账户同时提交任务时，严格按照提交顺序逐个求解，同一时间只运行一个任务。
- 排队页面实时展示前方任务数、队列任务数、预计开始时间和预计完成时间。
- 求解开始后自动切换为运行状态，完成后自动应用结果，无需用户重复提交。
- 根据近期实际求解耗时动态修正时间预估。
- 优化 OR-Tools 两阶段求解逻辑：先最大化已安排课时，再优化课程分布、优先日期与节次顺序，避免软规则降低排课完成率。
- 完善默认规则、课程规则、教师规则、连堂课、单双周、合班课、固定点和课程关系约束。
- 增加求解日志、规则命中说明和未排课程原因提示。

### 基础数据与任课信息

- 教师信息新增“周课时要求”，同步支持录入、导入和数据保存。
- 统一课程、教师和任课信息页面的模板下载、数据导入和数据导出操作。
- 优化课程安排，支持清理零课时课程并持久保存。
- 教学周次根据教学周期起止日期自动计算。
- 任课信息支持在线编辑、批量输入、模板导入、数据导出和表格下方保存。

### 排课规则设置

- 重构默认规则、课程规则、教师规则和权重分配的页面结构。
- 连堂课和课程时段规则支持按校区、年级、班级、课程筛选。
- 允许排课与禁止排课使用一致的方格交互，并提供规则统计与删除管理。
- 合班课支持勾选、全选、批量删除及二次确认。
- 去除无实际作用的范围字段和冗余操作入口。

### 排课方案与工作台

- 方案支持收藏置顶、完整复制、重命名和删除二次确认。
- 复制方案时同步复制排课结果，但复制后的课表保持未发布状态。
- 删除方案时同步提示并清理对应已发布课表数据。
- 优化班级与全年级锁定控件，统一选中、锁定和未锁定状态。
- 增加上午、下午节次视觉分隔，并降低分隔线的视觉重量。
- 优化排课进度展示，减少页面加载和数字变化造成的布局跳动。
- 完成 14 寸笔记本常见分辨率下的响应式适配。

### 课表管理与统计

- 每个方案只展示该方案最新发布的课表，同时保留方案切换。
- 班级、教师、课程和学校课表补充校区、年级和班级信息。
- 学校课表按年级合并表头，年级与班级分层展示。
- 新增按教学周筛选课表，单双周课程会根据所选周次分别显示。
- 支持课表打印、Excel 导出和显示字段配置。
- 新增教师课时统计，支持分页、字段排序和 Excel 导出。

### 使用体验

- 新增独立使用说明页面，按完整排课流程拆分介绍。
- 控制台增加使用说明入口。
- 数据加载统一使用骨架屏，降低页面内容突然出现的突兀感。
- 减少高饱和蓝色和非必要图标，优化页面密度与信息层级。
- Cloudflare 外网访问域名加入开发服务器允许列表。

### 验证

- 前端生产构建通过。
- NestJS 后端 TypeScript 构建通过。
- OR-Tools 求解器单元测试通过。
- 三个不同账户并发提交排课任务时，任务按提交顺序依次开始，测试期间最大求解并发数为 1。
- 1366 × 768 视口下完成排课工作台页面检查，无框架错误或控制台异常。

### 已知限制

- 当前智能排课队列保存在单个后端服务实例内。若后续部署多个后端实例，需要改用 Redis/BullMQ 等共享队列，才能继续保证跨实例的全局提交顺序。
- 后端服务重启后，尚未完成的内存排队任务不会恢复。
